# 布局

## 页签

::: demo
```html
<template>
  <zs-tabs :value.sync="active" :tab-list="tabList">
    <div v-show="active === 0">tab 1 content</div>
    <div v-show="active === 1">tab 2 content</div>
  </zs-tabs>
</template>

<script>
  export default {
    data() {
      return {
        active: 0,
        tabList: [
          { 
            name: 0,
            label: 'tab 1'
          },
          {
            name: 1,
            label: 'tab 2'
          }
        ]
      }
    }
  }
</script>
```
:::
